# redis的命令

## 通用命令

| 命令   | 描述                                     |
| :----- | :--------------------------------------- |
| keys   | 查看符合模板的所有key : keys *   keys a* |
| del    | 删除指定key                              |
| exists | 判断key是否存在                          |
| expire | 给key设置一个有效期                      |
| ttl    | 查看key有效期                            |
